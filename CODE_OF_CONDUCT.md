# Code of Conduct

This project adopts the [Contributor Covenant][cc], version 2.1, as its code of
conduct. The full text is available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

In short: be respectful and constructive. Harassment and exclusionary
behaviour are not tolerated, in any project space (issues, pull requests,
discussions) or in public spaces when representing the project.

## Reporting

Report unacceptable behaviour to the project maintainer at **atay@oezcan.me**.
All reports are reviewed and investigated promptly and confidentially. The
maintainer is obligated to respect the privacy and security of the reporter.

Enforcement follows the Contributor Covenant's
[enforcement guidelines][guidelines].

[cc]: https://www.contributor-covenant.org
[guidelines]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines
